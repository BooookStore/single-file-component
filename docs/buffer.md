# Vue.jsとCommonJS
Vue.jsをwebpackを使用してビルドすると、CommonJS等のルールに従ったモジュール管理をすることができる。なぜならwebpackがCommonJS等の依存関係解決を行うことができるからである。

[ドキュメント](https://webpack.js.org/concepts/modules/#what-is-a-webpack-module)
